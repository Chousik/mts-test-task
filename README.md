# MTS-test-task

## Описание проекта
В тестовом задание балы реализована автоконфигурация Spring Boot, которая добавляет HTTP‑заголовок `Foo: Bar` во все ответы HTTP-сервера Tomcat  
## Сборка и тестирование
```bash
./gradlew clean build
```
Команда соберёт проект и выполнит модульные тесты (`./gradlew test`)

## Использование автоконфигурации в другом приложении
1. Соберите артефакт с помощью `./gradlew fooHeaderAutoConfigJar`
2. Добавьте зависимость в целевой Spring Boot проект.
